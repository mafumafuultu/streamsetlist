## quick start

require: [nodejs](https://nodejs.org/en/download/)

1. start server

```bash
cd streamsetlist
npm i
npm run start
```

2. open `http://localhost:3000`
3. update `live.md`

live.md 
```md
<link rel="stylesheet" href="style.css">
<!--
class="u"

<div id="title" class="z" for free talk class</div>
-->
<div id="title" class="u"></div>

# Set list
* title / artist
```
